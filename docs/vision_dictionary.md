# MindReply Vision Dictionary

## Category

MindReply is an Executive Nervous System and Decision Infrastructure Layer.

It replaces the uncertain space between tools. The user gives MindReply a signal; MindReply returns one synthesis, one recommended action, and one silent memory update.

## Product Rules

- Every output contains one synthesis and one recommended action.
- High-risk communication pauses before execution.
- Memory stores patterns, not raw content.
- The interface stays inline and calm.
- The system removes a decision before it adds anything.

## Why A Cheap Copy Fails

The moat is not a screen. It is repeated private behavior: risk checks, redacted memory, signed receipts, and category language that makes the next action feel obvious.
